ComicCharacterNameSimpleNamedEntityRecognizer

This is a Qanary component meant for recognizing and annotating DBpedia's Comic Character entities within questions.

Example: "What is the real name of Catwoman?" In this case the name "Catwoman" can be found in the DBpedia knowledge base.
After recognizing the name as a DBpedia entity the component creates an annotation containing the start and end index of the recognized name within the question and stores it into the Qanary triplestore used for the current pipeline.
It does this by iterating through a list of comic character entities on DBpedia and checking if the question contains one of the names.

The component is meant to be used mainly for learning purposes and is designed to work in a question answering pipeline together with qanary_component-QB-ComicCharacterAlterEgoSimpleDBpediaQueryBuilder.

Although it was originally implemented as a Named Entity Recognizer it also offers functionality to store the resource link of named entities. A possible expansion for this component would be to include Named Entity Disambiguation.
