ComicCharacterAlterEgoSimpleDBpediaQueryBuilder

This is a Qanary component that creates SPARQL queries that can be used to find the real names of fictional characters in questions beginning with "what is the real name of".

Example: "What is the real name of Catwoman?"
Since the question begins with the specified prefix "what is the real name of", it looks for annotations for named Entities within this question that other components have stored in the triplestore. If such an annotation exists this component can find the recognized name and use it to generate the query that is the result of this process.
The query itself is relatively generic. However, based on the name found in the question the component adds a filter to the query, s.t., it only shows relevant results. This means that this component can only provide a possible solution, without any guarantee that the query will return an answer.

This query builder is meant to be mainly used for learning purposes and is designed to work in a Qanary Question Answering pipeline together with qanary_component-NER-ComicCharacterNameSimpleNamedEntityRecognizer.

As suggested by the name this component was originally built to generate queries for the DBpedia knowledge base. However, because it utilizes filters based on names and not on resource links it should be possible to expand this component to generate Wikidata queries too.
